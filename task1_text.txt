Практическая работа 1
Создание React-приложения. Работа с компонентами
Create React App (CRA) — это программа, которая помогает автоматически создавать готовые шаблоны React-приложений с уже настроенным этапом сборки и другими возможностями. По сути, это набор типовых npm-модулей и настроек для сборки большинства React приложений.
